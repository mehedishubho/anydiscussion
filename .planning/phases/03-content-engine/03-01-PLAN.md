## Phase Goal

**As a** member of the editorial team (admin/editor/author),
**I want to** write posts in a Tiptap v3 editor and save drafts with valid slugs and debounced autosave,
**so that** the body survives a sanitized JSON → HTML → render round-trip and the content-engine foundation is proven before taxonomy, media, and publishing layers build on it.

This is **Slice A** of the Phase-3 MVP vertical-slice plan: the post writing core. Every later slice (B: sanitization + taxonomy, C: media, D: publishing/preview) depends on the artifacts this plan ships — the editor extensions array (single source of truth for the SSR round-trip), the schema migration (preview token column + media provider renames that B/C consume), the slug/autosave/post-render utilities, and the actions/users.ts-style Server Action template.
