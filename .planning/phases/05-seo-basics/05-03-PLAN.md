## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Dashboard form input → Server Action → settings/post_seo table | Editor/admin-authored SEO strings cross from the client form into server-side storage; must be Zod-validated on both sides |
| settings table → saveSeoSettings → public metadata | Admin-authored site-wide SEO values render into public `<title>`/OG/JSON-LD; unauthorized writes = site defacement / canonical hijack |
| redirects table → not-found.tsx → permanentRedirect | A redirects row can redirect any unmatched path; a malicious new_path could be an open-redirect vector |
